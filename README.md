## 2. Instrucciones para Desarrollo e Implementación
**Paso 1:**
* En Github, crea un repositorio llamado **examen_julio-2025**

**Paso 3**
* **Solicitar a  [perplexity.ai](https://www.perplexity.ai/) lo siguiente:** (click derecho → "Abrir en nueva pestaña")

```txt
Necesito ayuda para investigar la educación del preceptor en la republica argentina, especificamente en la provincia de misiones. Dentro de que rama de la educación tecnica se encuentra, cual es el encuadre oficial, que se espera del perfil, hasta que edad es posible estudiar esta profesion, cuáles son las necesidades actuales. Recuerda que actualmente se están preparando preceptores sin requerimiento de titulo docente.
```
**Paso 4:**
* Abre el Procesador de texto de google y crea un archivo con el nombre **Educa_PRECE_Misiones**.
*  [[pulsa aquí para abrir el Procesador de Texto](https://docs.google.com/document/u/0/)] (click derecho → "Abrir enlace en nueva pestaña")
  * _Recuerda habilitar markdown dentro de preferencias_ 
  * Con la información generada por perplexity.ai, **Copia y pega lo generado por perplexity** 

_Debes respetar la siguiente consigna_
> * Primer página debe ser la Portada.
> * Segunda página estará la Tabla de Contenido o Indice.
> * Tercera cuerpo del documento
> * Ultima hora del documento realiza un salto de sección y pega la información contenida del siguiente enlace [[pulsa aquí para abrir el archivo en Procesador de Texto](https://docs.google.com/document/d/1qKwgICQN0zXG3mh0tTTgJaP99iXmJwpdT6WuHCYLwRA/edit?usp=sharing)] (click derecho → "Abrir enlace en nueva pestaña")


* Separa en 4 Secciones:
> * Sección 1, Portada
> * Sección 2, Indice 
> * Sección 3, Cuerpo del documento
> * Sección 4, Tabla (PAGINA HORIZONTAL)
> * Sección 5, Resto del documento

* Encabezado, pie de pagina, Notas al pie
> * Recuerda colocar el encabezado y pie de pagina en los lugares indicados.
> * Recuerda armar la tabla de contenido y la nota al pie.

**Paso 5:**
> * Abre [gamma.app](https://gamma.app/create/paste) (click derecho → "Abrir en nueva pestaña") y pega tu reporte para generar una presentación de 4 paginas.
> * Recuerda compartir el link para pegar en el siguiente paso.

**Paso 6:**
* En tu github, abre y edita el README.MD
* Copia lo siguiente y modifica donde te solicita para que se vean tus archivos generados:
*  ```txt

      # LIBRES/REGULARES
